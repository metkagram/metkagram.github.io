# Metkagram brand assets

## Recommended files

- `logo/metkagram-logo-light.svg` — primary logo for light backgrounds.
- `logo/metkagram-logo-dark.svg` — logo for dark backgrounds.
- `icons/metkagram-mark.svg` — scalable compact mark.
- `icons/favicon.ico` — browser favicon.
- `icons/metkagram-icon-180x180.png` — Apple touch icon.
- `icons/metkagram-icon-192x192.png` — PWA icon.
- `icons/metkagram-icon-512x512.png` — large PWA/app icon.
- `social/metkagram-social-preview-1200x630.png` — Open Graph / LinkedIn preview.
- `web/head-snippet.html` — ready-to-copy HTML tags.
- `web/site.webmanifest` — basic web app manifest.

## Brand colors

- Yellow: `#FFC400`
- Black: `#111111`
- Off-white: `#F7F7F3`
- White: `#FFFFFF`

## Usage

Keep clear space around the logo equal to roughly half the height of the yellow tag. Do not stretch, rotate, add shadows, or alter the angle-bracket construction.

The `source/` folder contains editable SVG versions with live Inter text. Public SVG files in `logo/`, `icons/`, and `social/` have text converted to paths and do not require the font.
